# O Caiçara - Bairros de Santos
![O Caiçara](https://github.com/hericmr/ocaicara/blob/main/screenplay.png?raw=true)

![Assista ao vídeo da gameplay]([https://www.youtube.com/watch?v=iNLuEgj2kLI](https://youtu.be/iNLuEgj2kLI?si=gFTgwhcd1zR2g7JI))


Este é um jogo desenvolvido em Python usando Pygame, onde o objetivo é encontrar biqueiras e bairros na cidade de Santos, São Paulo.

## Visão Geral

Explore o charme de Santos e ajude o nosso Marginal Alado a fazer seu corre. Encontre as biqueiras e os diferentes bairros da cidade, testando não apenas sua malandragem, mas também seu conhecimento geográfico. Prepare-se para uma aventura emocionante, onde sua sagacidade e astúcia serão postas à prova nos becos e vielas da cidade.

## Requisitos

Para executar este jogo, certifique-se de ter Python e Pygame instalados em seu ambiente. Você pode instalar os requisitos executando o seguinte comando:

```bash
pip install -r requirements.txt
```

## Como Jogar

1. **Clone este repositório:**

```bash
git clone https://github.com/hericmr/ocaicara.git
```

2. **Navegue até o diretório do projeto:**

```bash
cd ocaicara
```

3. **Instale os requisitos:**

```bash
pip install -r requirements.txt
```

4. **Execute o jogo:**

```bash
python main.py
```

5. **Divirta-se jogando!**
